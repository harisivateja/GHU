Proposed Solution

A comprehensive platform for predictive student retention and proactive support

The Centralized Student Dashboard is a full-stack web application designed to be the single source of truth for student performance and risk assessment at Horizon State University. Built on modern React, FastAPI, and MongoDB architecture, it consolidates disparate data streams into one actionable platform that shifts advising from reactive remediation to proactive support.
