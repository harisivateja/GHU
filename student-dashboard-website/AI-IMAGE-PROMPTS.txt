═══════════════════════════════════════════════════════════════
IMAGE GENERATION TIPS
═══════════════════════════════════════════════════════════════

DALL-E 3 Settings:
- Quality: Standard or HD
- Style: Natural (for realistic) or Vivid (for stylized)
- Aspect Ratio: Adjust based on recommended size above

Midjourney Settings:
- Add parameters: --ar 16:9 (for wide images) or --ar 4:3 (for standard)
- Add: --style raw (for photorealistic) or --stylize 500 (for artistic)
- Add: --quality 2 (for higher detail)

Stable Diffusion:
- Steps: 30-50
- CFG Scale: 7-10
- Sampler: DPM++ 2M Karras or Euler a
- Model: Stable Diffusion XL for best quality

Post-Processing:
- Optimize file size using tinypng.com or squoosh.app
- Target: <500KB per image for fast web loading
- Convert to WebP format for even smaller sizes (optional)
- Add slight blur to backgrounds if using as hero images
